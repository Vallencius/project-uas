LANGKAH INSTALASI
- Clone repository
- Buka CMD
- Masuk ke directory project-uas
- npm install
- ng serve
- buka http://localhost:4200/

Jika instalasi diatas error, mungkin dibutuhkan beberapa instalasi lain seperti dibawah:
- npm install firebase @angular/fire
- ng add @angular/material
